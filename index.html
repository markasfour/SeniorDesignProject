    What is GTWENDS

    GTWENDS is an interactive map of Google and Twitter trends.

    Overview of technologies used:

    • Web crawling
    • Twitter API
    • Cassandra
    • Hadoop
    • AngularJS

    Software Development Cycle: Evolutionary Prototyping Model

    Functionality of GTWENDS

    View trends from Twitter and Google on a heat map of the US

    • Filter specific trends
    • Filter from origin
    • Filter by time
